How to use Time2eat online cookbook:

Looking for recipes
  1. To get an idea of the number of recipes in the website split by difficulty, category and cuisine go to the statistics page to see the recipes dashboard.
  2. In the home page you can see all the recipes sorted by number of votes. Use the carousel to see all af them.
  3. At the bottom of the home page you have the filter section that you can use to display only the recipes matching your criteria. Not all the filter options in the filter are required. (i.e. you can filter only by cuisine or you can filter by category, cuisine, allergens and difficulty)
  4. Click on the recipe image or on the link "view image" to get detailed inforamtion about the recipe
  5. If you are not the author of the recipie you can vote it.
  6. If you are the author of the recipe you will be able to edit it or remove it.
Adding new recipes
  1. Go to "Manage categories" page and if the category or cuisine for your recipe is not included in the list add it.
  2. Go to the add recipe page and add the recipe.
