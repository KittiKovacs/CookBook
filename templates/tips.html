How to use Time2eat

home
View recipes

In the home page you can see all the recipes. They are sorted by number of votes being the most voted on the top of the carousel of images and then they are sorting on descending order if you move to right and on ascending order if you move to left.

Below the carousel you can find the filter section where you can filter the recipes based on one or more than one criteria. (i.e you could filter only by cuisine or you could filter by cuisine, type of meal, allergens to exclude and difficulty)

trending_up
Recipes dashboard

In this section you can have an overview of the recipes included in the app. You can see the number of recipes split by different criteria and also a summary table with some more details.

settings
Manage type of meal / cuisines

In this section you can see all the type of meals and cuisines that already exist in the app.

If you are planning to add recipes and the type of meal or cuisine is not included in the list make sure you add it before starting adding a new recipe.

add_box
Add recipe

Use this section to enter your recipes.

* In order to get more votes in your recipes we advice you to include a picture for the recipe but it is not compulsory. If you don't have an image we will include a default picture.
